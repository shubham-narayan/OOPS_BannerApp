# OOPS Banner App

This project demonstrates Object-Oriented Programming concepts using a console-based banner application.

The application evolves through multiple use cases (UC1 – UC8), gradually improving the design and structure.

## Use Cases

UC1 – Print "OOPS" in console  
UC2 – Display OOPS in banner format using `*`  
UC3 – Refactor code using functions  
UC4 – Introduce a class to represent banner characters  
UC5 – Store character patterns in arrays  
UC6 – Use a map to store character patterns  
UC7 – Improve modularity  
UC8 – Final refactoring and clean design

## Language

C++

## Run

Compile and run:

```
g++ src/BannerApp.cpp
./a.out
```
